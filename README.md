# Lab 2: Predictive Analytics with Machine Learning

Intro To AI — supervised regression, multi-class classification, and K-Means clustering on NYC Yellow Taxi and obesity-level datasets.

## Setup

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

pip install -r requirements.txt
jupyter lab
```

Open `lab_2_predictive_analytics.ipynb` and run all cells (Kernel → Restart & Run All).

Datasets are loaded directly from the URLs provided in the notebook; no local download is required.

## Contents

| Section | Task | Dataset |
|---------|------|---------|
| 1 | Regression (`tip_amount`) | NYC Yellow Taxi |
| 2 | Multi-class classification (`NObeyesdad`) | Obesity levels |
| 3 | K-Means clustering | Obesity features (labels hidden) |
